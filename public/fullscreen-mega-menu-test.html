Тест полноэкранного мега-меню

Новое полноэкранное мега-меню включает:
  • Полноэкранный режим (кроме шапки)
  • Двухколоночная структура (каталоги + контент)
  • Поиск по каталогам и категориям
  • Хлебные крошки для навигации
  • Клавиатурная навигация (Escape, Enter, стрелки)
  • Адаптивность для мобильных устройств

Как использовать

  1. Нажмите на кнопку "Каталоги товаров" в навигации
  2. Выберите каталог в левой колонке
  3. Просматривайте группы и категории в правой части
  4. Используйте поиск для быстрого поиска
  5. Нажмите Escape или кнопку закрытия для выхода

Горячие клавиши

  • Escape - закрыть меню
  • Enter - выбрать каталог
  • ↑↓ - навигация по каталогам
  • Tab - навигация по элементам
Мобильная версия

На мобильных устройствах мега-меню адаптируется:

  • Каталоги отображаются горизонтально в верхней части
  • Контент занимает всю оставшуюся высоту
  • Поиск остается доступным
  • Свайп-жесты для навигации
